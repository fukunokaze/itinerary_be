# Travel Planner App Backend App

## Bussiness Requirements
- Will serve the backend of Travel Planner app
- Should be able to manage record of Trip where it contains Flight, Lodging, and Activity/Trip Event


## Technical Spec
- Should be written in latest .NET
- Use the latest and secure third party libraries if necessary
- Follow SOLID principle
- WebAPI part should not contain any bussiness logic or direct database call
- Public methods in logic service should have unit test.
- Unit test naming should follow NameOfMethod_Scenario_ExpectedResult format
